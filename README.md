# mac

Execute the following command to set up your new Mac:

> [!IMPORTANT]
> [install.sh](install.sh) assumes that you have copied your `.ssh` folder into `$HOME` and that it contains three files; `config`, `rajdeepsh`, and `rajdeepsh.pub`.

```bash
curl -fsSL https://raw.githubusercontent.com/rajdeepsh/mac/main/install.sh | bash
```
